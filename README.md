# clinical-reference-mcp

[![License](https://img.shields.io/badge/license-Apache%202.0-blue.svg)](LICENSE)

> **Status: pre-alpha — no tools implemented yet.** Phase 1 of 5: project
> scaffolding, types, and safety primitives only. The MCP server, transports,
> tools, API clients, and CLI do not exist yet. Do not install or use.

An MCP (Model Context Protocol) server exposing drug, prescription, and pharmacology
reference tools sourced from authoritative public APIs (openFDA, RxNorm/RxNav).
The first usable release will be v0.1.0.

> **Developer reference, not clinical use.** When complete, this server will wrap
> publicly published regulator data so AI engineers can ground their tools in
> authoritative sources. It does not and will not give medical advice and must
> not be used for clinical decision-making.

Full documentation will arrive with v0.1.0. See [LICENSE](LICENSE) and [NOTICE](NOTICE)
for licensing and attribution.
